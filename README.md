# AI Journey

This repository documents a practical learning path in AI, automation, and AI-assisted coding.

## Focus

- Building useful AI workflows for real business tasks
- Learning automation tools such as no-code workflow builders, APIs, and data pipelines
- Practicing AI-assisted coding through small desktop apps, browser tools, and portfolio projects
- Turning experiments into clear demos, case studies, and reusable project notes

## Current Direction

The main goal is to move from general AI exploration to practical systems that process input, apply AI reasoning, save structured results, and produce clear next actions for a human user.

## Project Principles

- Keep projects small enough to finish
- Build visible artifacts, not only notes
- Prefer simple workflows before complex architectures
- Document what was built, why it matters, and what can be improved next

## Repository Scope

This is a public-safe folder created specifically for GitHub. It does not include private notes, personal documents, credentials, client data, or local Obsidian system files.
